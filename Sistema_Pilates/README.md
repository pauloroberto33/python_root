# Sistema Pilates Studio Web

Este projeto converte o código original em um sistema web profissional usando FastAPI.

## Como usar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o sistema:
   ```bash
   python run.py
   ```

3. O navegador abrirá automaticamente em `http://127.0.0.1:8000`.

## Recursos

- Cadastro de clientes
- Registro de avaliações
- Registro de pagamentos
- Controle de despesas
- Dashboard financeiro com visão de receitas e despesas
- Dados de teste já carregados no banco SQLite
